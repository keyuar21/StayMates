# 🏠 Staymates: Roommate Finder Platform  

Staymates is a web-based platform designed to connect individuals seeking roommates. Think of it as a social media application, but specifically tailored to help users find ideal living companions. The platform integrates user-friendly features like authentication, profile management, friend lists, and real-time interactions, making it a comprehensive solution for shared living arrangements.

---

## 🚀 Features  
### 🔑 **User Authentication**  
- Secure login and registration with session management.  

### 👤 **Profile Management**  
- Users can create detailed profiles, including:  
  - Profile picture uploads.  
  - Social media links (e.g., Twitter, Instagram).  
  - Personal details like age, university, preferences (e.g., smoker/non-smoker).  

### 🤝 **Friend List Functionality**  
- Add/remove friends with pending requests and accepted lists.  
- Real-time interaction with friends, including messaging functionality.  

### 🏠 **Host a Property**  
- List properties for potential roommates with the following details:  
  - Property type (flat, house, or PG).  
  - Pet and smoking permissions.  
  - Multiple image uploads with previews.  
  - Rent options (per day, week, month).  

### 🌐 **Explore Page**  
- View all registered users and their profiles.  
- Add users to your friend list with just one click.  

### 💬 **Messaging System**  
- Communicate with friends through a secure, real-time messaging feature.  

---

## 💻 Technologies Used  
### **Frontend**  
- **HTML**, **CSS**, **EJS** for dynamic and responsive UI.  

### **Backend**  
- **Node.js**, **Express.js** for routing and server logic.  

### **Database**  
- **PostgreSQL** for data management and storage.  

### **Tools**  
- **Multer** for image uploads.  
- **pgAdmin** for database administration.  
- **GitHub** for version control.
